# CSS Selectors Assignment

This project demonstrates the use of different CSS selectors such as element selectors, class selectors, ID selectors, descendant selectors, and pseudo-class selectors. The webpage contains headings, paragraphs, links, and lists styled using an external CSS file.

## Features

- Styling using class selectors (`.head`)
- Styling using ID selectors (`#bglime`, `#para`)
- Element selector styling for paragraphs
- Styling specific text using the `span` tag
- Custom styling for hyperlinks without using inline CSS
- Different colors applied to list items using CSS selectors
- Separation of structure (HTML) and styling (CSS)

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download or clone the project.
2. Ensure both `index.html` and `style2.css` are in the same folder.
3. Open `index.html` in any web browser.
4. The styled webpage will be displayed.

## Learning Objectives

- Understand CSS selectors
- Apply styles using classes and IDs
- Use external stylesheets
- Style links and list items without inline CSS
- Practice clean and structured web development

## Project Files

- `index.html` – Contains the webpage structure.
- `style2.css` – Contains all styling rules.
